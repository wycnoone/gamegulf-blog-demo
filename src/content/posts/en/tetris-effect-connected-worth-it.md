## What kind of game is Tetris Effect: Connected, really?

**Tetris Effect: Connected** is **Enhance’s “concert Tetris”**—think **Lumines-adjacent rhythm spectacle** applied to **Guideline Tetris**, built by the **Resonair / Monstars** team that shipped **Rez Infinite** and **Lumines Remastered**.

1. **Journey** strings **30+ stages** with bespoke audio-reactive skins—each map teaches new visual language while keeping the same hard-drop muscle memory.
2. **Zone** freezes incoming pieces so you can erase a stacked threat or convert clears into huge bursts—a different risk profile than **Tetris 99** sprint rules.
3. **Connected** stitches **three boards** together for **cooperative boss raids**, while **Zone Battle**, **Score Attack**, and **Friend Match** (+ **spectators**) cover duels without relying on a generic “Classic only” mode list.

If you already bounce off **reactive backgrounds**, the campaign still might not click; if you love **audio-visual feedback loops**, this is the flagship Switch Tetris for that niche.
